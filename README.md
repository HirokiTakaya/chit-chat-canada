# Chit-Chat Canada 🍁

A bilingual community homepage for Chit-Chat Canada — Vancouver's English & Japanese language exchange meetup.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Production

```bash
npm run build
npm run start
```

## Tech Stack

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Google Fonts** (Outfit + Playfair Display)
